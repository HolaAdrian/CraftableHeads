CraftableHeads

A simple and flexible plugin that lets players craft and customize player heads using texture URLs.
Includes fully configurable crafting recipes and messages via config.yml.

âœ¨ Features

Craft player heads using your own custom crafting recipe (fully configurable in config.yml).

Change the texture of any player head using a simple command.

Built-in validation and helpful error messages for invalid texture links.

Highly customizable messages with support for color codes.

Works on most Minecraft versions.

Supports Paper, Spigot, and CraftBukkit servers.
